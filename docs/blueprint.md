# **App Name**: ChitFund Pro

## Core Features:

- Admin User Authentication: Secure admin login using Firebase Authentication.
- Member Management: Add new members with their details (name, phone, join date, monthly amount) and manage their active/inactive status. View and edit member profiles.
- Payment Tracking & Recording: Record monthly payments for each member, specifying amount, date, and payment status. View a comprehensive payment history for all members.
- Financial Overview & Reports: Display total collected amounts, visualize pending payments, and generate summaries for financial reporting.
- Chit Round & Winner History: Track chit rounds, record winning member's name and winning amount for each round. Display a historical record of all chit winners.
- Responsive Admin Dashboard: A centralized dashboard for administrators to quickly view key metrics and navigate to different sections of the application. The UI will be mobile-responsive.

## Style Guidelines:

- Primary color: Professional deep blue-grey (#36516E), evoking trust and organization. This is a muted, versatile shade.
- Background color: A very light, desaturated blue-grey (#EFF2F5), providing a clean and understated canvas that maintains visual coherence with the primary color.
- Accent color: A vibrant, clear sky blue (#38BBDB), providing a crisp contrast for calls-to-action and important highlights without clashing.
- Headline and body font: 'Inter', a modern sans-serif typeface known for its neutrality, excellent readability, and versatility across various text sizes and weights, ideal for data-intensive interfaces.
- Utilize clean, functional, and vector-based icons that clearly communicate their purpose. Prioritize consistency in style and line weight to maintain a professional aesthetic suitable for a management tool.
- Implement a clean, organized, and responsive layout featuring intuitive navigation. Data tables will be structured for easy readability across desktop and mobile devices, optimizing for clarity and efficient information scanning.
- Incorporate subtle and functional animations for user feedback, such as confirmation of data submission, seamless page transitions, and smooth sorting/filtering actions within data tables. Animations should be swift and enhance usability without causing distractions.
